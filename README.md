### 알고리즘👀

#### 커밋 규칙

- commit msg : [문제 출처(플랫폼)] 문제이름 / 난이도 / 걸린시간
- description : 문제 주소(option)
- 터미널에서 작성법 :

```js
git commit -m "[BOJ] Hello World / 브론즈5/ 1분" -m "https://www.acmicpc.net/problem/2557"
```

- 플랫폼 작성법 통일 :
  - [BOJ] - 백준
  - [PGS] - 프로그래머스
  - [SEA] - 삼성SW Expert Academy
  - [ETC] - 그 외
